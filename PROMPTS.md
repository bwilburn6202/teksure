- PR Prep: "Provide a 1-page patch plan for a given feature, including file diffs, rationale, and test steps."
- Quick Fix: "List the minimal changes to fix [bug/issue] with patch diff."
- Accessibility Audit: "Produce a 20-point accessibility checklist for TekSure with priority and fixes."
- Performance Sprint: "Outline a 4-hour performance improvement sprint with measurable metrics (TTFB, LCP, CLS)."
- Data Model Review: "Review current data models (guides, progress, ratings) and propose 3 concrete migrations."
- Security Spin: "Identify the top 6 security gaps in TekSure and give precise mitigations tailored to a React + Supabase stack."
- Testing Plan: "Draft a minimal test plan targeting 60%+ coverage for critical modules with unit and integration tests."
- Content Strategy: "Propose a US-centric content plan with 6 new guides, 2 video walkthroughs, and a refresh schedule."
