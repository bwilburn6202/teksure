# Source Validation — TekSure

## Purpose
Ensures all information in TekSure guides comes from trustworthy, authoritative sources.

## When to Use
- Before adding any external link or reference to a guide
- During content freshness checks
- When researching new guide topics

## Source Tiers
### Tier 1 — Always trusted
- Apple Support (support.apple.com)
- Google Support (support.google.com)
- Microsoft Support (support.microsoft.com)
- Samsung Support (samsung.com/us/support)
- FTC (ftc.gov, consumer.ftc.gov)
- CISA (cisa.gov)

### Tier 2 — Trusted with verification
- How-To Geek, Tom's Guide, AARP, GCFGlobal
- Official YouTube channels (@AppleSupport, @Google, @Microsoft, @Samsung)
- Senior Planet, TechBoomers, Digital Unite

### Tier 3 — Use with caution
- Reddit (verify claims independently)
- Quora (verify claims independently)
- Community forums (Apple Communities, Microsoft Answers)

### Never use
- Random blogs with no author attribution
- AI-generated content farms
- Sources older than 2 years for tech content
- Sites with excessive ads or clickbait

## Validation Checklist
- [ ] Source is from Tier 1 or Tier 2
- [ ] Information is current (published within last 12 months)
- [ ] Author or organization is identifiable
- [ ] Claims can be cross-referenced with another source
- [ ] URL is live and accessible

## Constraints
- Never link to sites that require payment to access
- Never present forum opinions as verified facts
- Always prefer official sources over third-party explanations
