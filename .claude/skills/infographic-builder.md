# Infographic Builder — TekSure

## Purpose
Turns guide content into visual-friendly formats for sharing, printing, or embedding.

## When to Use
- Creating shareable safety checklists
- Building printable quick-reference cards
- Designing comparison tables (iPhone vs Android)
- Summarizing multi-step processes visually

## Infographic Types
1. **Numbered steps:** Icon + step title + one-line description
2. **Checklist:** Checkbox items for safety/setup tasks
3. **Comparison table:** Two columns, feature by feature
4. **Statistics:** Key numbers with context
5. **Timeline:** Before → During → After process flows

## Output Format
- Title and subtitle
- Sections with headings
- Bullet points or numbered items
- Icon suggestions for each section
- Color scheme recommendation

## TekSure-Specific Rules
- Designed to be readable when printed at standard letter size
- Large font sizes (minimum 14pt equivalent)
- High contrast colors
- No more than 7 items per infographic
- Include TekSure branding and URL

## Constraints
- Avoid excessive text — visuals should carry the message
- Keep it to one "page" equivalent
- Must be accessible (not image-only — include text version)
