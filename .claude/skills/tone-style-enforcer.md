# Tone & Style Enforcer — TekSure

## Purpose
Keeps all TekSure content consistent with a warm, patient, plain-English voice designed for non-technical users and seniors.

## When to Use
- Before publishing any new guide, tip, scam alert, or page copy
- During weekly/monthly content audits
- When reviewing auto-generated content

## Voice Rules
- Plain English always — if a 12-year-old can't understand it, rewrite it
- No jargon without explanation — if you must use a tech term, explain it in parentheses
- Active voice — "Tap the Settings icon" not "The Settings icon should be tapped"
- Direct address — "You" not "the user" or "one"
- Encouraging, never condescending — "Here's how" not "Simply do this"
- American English — color not colour, organize not organise

## Banned Words
Never use: simply, just, easy, obviously, leverage, utilize, seamless, cutting-edge, robust, scalable, ecosystem, paradigm, synergy, deep dive, circle back, touch base

## Preferred Alternatives
| Instead of | Say |
|-----------|-----|
| Navigate to Settings | Open Settings / Go to Settings |
| Authenticate | Sign in / Log in |
| Configure | Set up |
| Terminate | Close / Stop |
| Execute | Run / Start |
| Functionality | Feature |
| Interface | Screen |
| Credentials | Username and password |
| Connectivity | Connection / Internet |
| Pro Tip | Quick Tip |

## Tone by Context
- **Guides:** Warm, patient, step-by-step. Like explaining to a friend over coffee.
- **Scam alerts:** Urgent but empowering. "Here's how to protect yourself" not "You're in danger."
- **Tips:** Conversational, practical. One thing you can do TODAY.
- **Error messages:** Reassuring. "Something went wrong, but don't worry."
- **News:** Explain WHY it matters to a regular person.

## Quality Checklist
- [ ] No unexplained jargon
- [ ] Sentences under 25 words where possible
- [ ] Steps start with an action verb (Tap, Open, Click, Go to)
- [ ] No passive voice in instructions
- [ ] Read aloud — does it sound natural?

## Constraints
- No deviation from selected tone
- Maintain clarity over cleverness
- Never make the reader feel stupid
