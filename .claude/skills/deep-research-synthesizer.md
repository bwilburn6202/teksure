# Deep Research Synthesizer — TekSure

## Purpose
Converts large amounts of research (web searches, forum posts, official docs) into structured, actionable insights for guide creation.

## When to Use
- Weekly guide enrichment research phase
- Investigating new tech problems to cover
- Gathering data for competitive intelligence

## Instructions
1. Gather data from multiple sources (official support sites, forums, news)
2. Filter out low-quality, unverified, or outdated information
3. Identify patterns — what problems come up repeatedly?
4. Synthesize into actionable takeaways
5. Rank by impact (how many people does this affect?)

## Source Priority (highest to lowest)
1. Official manufacturer support (Apple, Google, Microsoft, Samsung)
2. Government safety sites (FTC, CISA, IC3)
3. Reputable tech publications (How-To Geek, Tom's Guide, AARP)
4. Community forums (Reddit, Apple Communities, Microsoft Answers)
5. Everything else — verify before using

## Output Format
- Key findings (bullet points)
- Recommended actions (what to write/update)
- Source URLs for reference
- Confidence level (high/medium/low)

## Constraints
- Never present unverified claims as fact
- Always note the source
- Focus on what's actionable for TekSure's audience
