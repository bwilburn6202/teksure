# Content Repurposing Engine — TekSure

## Purpose
Transforms full-length guides into multiple content formats for different channels while maintaining TekSure's plain-English voice.

## When to Use
- After publishing new guides (bi-weekly content-repurposing task)
- When preparing newsletter content
- When creating social media campaigns

## Output Formats

### Social Media Post (Twitter/Facebook)
- Hook line (under 100 chars, curiosity-driven)
- 2-3 bullet takeaways
- Call to action: "Full guide at TekSure.com"
- No hashtag spam — max 2 relevant hashtags

### Newsletter Snippet
- 2-3 sentences: hook + key takeaway + link prompt
- Example: "Did you know scammers can now clone your family's voice with AI? Here's how to protect yourself with one simple family agreement. Read the full guide →"

### Email Tip
- Subject line (under 50 chars)
- One paragraph, one action
- Link to full guide

### Short Video Script (60-90 seconds)
- Hook (5 sec): state the problem
- Walkthrough (50-70 sec): screen-by-screen steps
- Recap (10 sec): what you accomplished + CTA

## TekSure-Specific Rules
- Maintain plain-English voice across ALL formats
- Never use fear-based hooks for scam content — use empowerment
- American audience context
- Always link back to the full guide on TekSure.com

## Constraints
- Preserve core meaning from original guide
- No verbosity — every word must earn its place
- Format must match channel conventions
