# TekSure Content Strategy

Monthly content strategy documents for TekSure.com — a tech support and digital literacy platform for non-technical users and seniors.

## Purpose

These strategies work hand-in-hand with the TekSure Development Roadmap to coordinate content creation with feature development. Each monthly strategy includes:

- Market research on what seniors and everyday users are searching for
- Gap analysis against TekSure's existing guide library
- Detailed outlines for new guides to fill those gaps
- Content calendar with publishing order
- Roadmap alignment showing how new content connects to site features
- Category health checks and strategic recommendations

## How to Use

1. At the start of each month, generate the content strategy for that month
2. Cross-reference with the roadmap (`teksure-roadmap-strategy.md`) to align guides with upcoming features
3. Use the guide outlines to build actual guides in `src/data/guides.ts`
4. Track publishing against the content calendar
5. Roll unfinished guides forward into the next month's strategy
